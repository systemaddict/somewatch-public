Privacy Policy for Link Text Extractor

Overview

Link Text Extractor is a browser extension that helps users extract and save readable text content from web pages. We take your privacy seriously and want to be transparent about how this extension works.

Data Collection and Storage

  • All extracted text is stored locally in your browser using Chrome's local storage
  • No data is ever sent to external servers
  • No personal information is collected
  • No analytics or tracking is implemented

Permissions Used

This extension requires certain permissions to function:

  • Access to webpage content: Required to extract text from pages you choose
  • Local storage: To save extracted articles on your device
  • Tab access: To perform text extraction in background

Data Removal

  • All stored data can be deleted at any time using the "Clear All" button in the extension
  • Uninstalling the extension will remove all stored data

Updates

This privacy policy may be updated to reflect changes in the extension's functionality.
